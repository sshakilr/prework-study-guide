Prework Study Guide

HTML

  1. HTML, in full hypertext markup language, a formatting system for displaying material retrieved over the Internet.
  2. Each retrieval unit is known as a Web page (from World Wide Web), and such pages frequently contain hypertext links that allow related pages to be retrieved.

CSS

  1. CSS stands for Cascading Style Sheets
  2. This style sheet language is used for defining the presentation of a document written in some markup language like HTML
  3. CSS has now become a very important part of building blogs and websites.
